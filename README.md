# EmbedKit
EmbedKit is a configurable toolkit for working with embeddings in real applications.

It abstracts the messy parts of embedding workflows — chunking, overlap, normalization, vector search, projections, and indexing — while keeping the underlying pieces accessible for people who want deeper control.

Use it at three levels:
- Simple API to chunk, embed, and search quickly
- Configurable workflows to tune behavior without rewriting plumbing
- Low-level interfaces to swap internals and build custom pipelines

EmbedKit is built for developers who want the power of embeddings without the framework sludge.
